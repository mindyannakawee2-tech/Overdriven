# Overdriven

Overdriven is parappa the rapper inspired game built with ***Scratch3*** & ***ovrCRTdata***

# Story
You'll be playing as Rhythia, a girl who wants to know the secret of the ***COLA*** company.

# Releases Date
***Planned*** on 1st January 2027 ***Final Release*** not enough progress to tell

# Controls
***Keyboard*** Q, W, E, A, S, D, up arrow, down arrow, left arrow, right arrow

***Xbox Controller*** L, Y, R, X, A, B, joyPad up, joyPad down, joyPad left, joyPad right [Not Available Yet]
